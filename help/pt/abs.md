* `Z = abs(X)`

Compute the magnitude of `X`.

The magnitude is defined as %%abs(Z) = sqrt(x^2 + y^2)%%.

For example:

>> `abs(3 + 4i)`

>> %%abs(3+4i)=5%%

See also: `arg`.
