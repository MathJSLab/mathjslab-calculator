* `TF = isrow (X)`

Return `true` if `X` is a row vector.

A row vector is a 2-D array for which `size (X)` returns %%[1, N]%% with non-negative %%N%%.

See also: `iscolumn`, `isscalar`, `isvector`, `ismatrix`, `size`.
